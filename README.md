# Forecrest

Financial simulation dashboard for startups — model revenues, operating costs, equity, and profitability with real-time projections.

**Forecrest** (Forecast + Crest) helps startups get a clear, simulated financial overview: ARR, runway, cap table, burn rate, and more.

---

## Stack

- **React 18** + **Vite 6**
- **GSAP** animations
- **Phosphor Icons**
- **Vitest** testing
- **i18n** (FR/EN)
- **Dark mode** with animated toggle

## Quick Start

```bash
npm install
npm run dev        # Dev server
npm run build      # Production build
npm test           # Run tests
```

## Docker

```bash
docker build -t forecrest .
docker run -p 3000:80 forecrest
```
